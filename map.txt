Backend
=====================================
/controllers                - A C-part of MVC-pattern
/inits                      - Express init params includes
/libs                       - Global utilities for App
/models                     - A M-part of MVC-pattern
/public                     - Frontend static files
/routes                     - Express routes includes
/views                      - A V-part of MVC-pattern

Frontend (root of /public dir)
=====================================
/flat-ui                    - A Bootstrap-based CSS framework
/javascripts                - JS-files dir
/javascripts/extensions     - app.js core extensions
/javascripts/libs           - JS-libs dir (jQuery, other frameworks, etc.)
/javascripts/modules        - App modules
/javascripts/plugins        - Jquery plugins folder
/stylesheets                - CSS-files dir
/templates                  - A frontend template engine html's folder
